# Movies 

## Dev

1. Clonar el repositorio
2. Instalar las dependencias con `npm install`
3. Clonar el archivo `.env.example` y renombrarlo a `.env` y completar las variables de entorno
4. Correr el proyecto con `npx react-native run-android` o `npx react-native run-ios` configurando el emulador correspondiente.
